- UI Activity HUB: <br/>
https://github.com/SVProgressHUD/SVProgressHUD <br/>
- Network Layer: <br/>
https://github.com/Moya/Moya <br/>
- Image Download/Cache: <br/>
https://github.com/onevcat/Kingfisher <br/>

Open 'JobSityiOSChallenge.xcworkspace' file<br/>
RUN!<br/><br/>
